# AFC Card Studio

Structure simplifiée du projet :

- `index.html` : point d’entrée de l’application.
- `assets/css/style.css` : styles de l’éditeur et de la carte.
- `assets/js/script.js` : logique de génération des stats et export.
- `assets/data/rarity.json` : données de rareté.
- `assets/images/` : dossier réservé aux images (vidé des `.jpg` demandés).

## Lancer en local

Ouvre simplement `index.html` dans ton navigateur.
